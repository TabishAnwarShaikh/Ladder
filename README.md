Snakes and Ladders Games.
----------------------------

Pre-Requisite
---------------------------
1. Maven 3x
2. JDK 1.7
3. Git client.

Installation:
-----------------------------
1. Checkout the repository in a ${dir}
2. Run mvn clean install 
3. To run the program go to the ${dir}/target folder
4. Type in java -jar snakesnladders-0.0.1-SNAPSHOT-javadoc.jar
